# Product Requirements Document
## College Tech Fest Website

## 1. Product Overview
The system is a responsive web platform for promoting a college technical festival.  
The website will present event information, schedules, speakers, competitions, and registration details.

The design and layout will closely replicate the UI style of the SREC Utsava website.

## 2. Goals
- Deliver a visually engaging event website
- Provide easy navigation for event information
- Showcase competitions and workshops
- Enable participants to register for events
- Demonstrate modern web UI capabilities

## 3. Target Users
Primary users include:

Students
Participants
Event organizers
Guest speakers

## 4. Core Features

### Hero Section
Displays:
- Fest name
- Tagline
- Date and venue
- Registration button

### Statistics Section
Displays key numbers such as:
- Total events
- Participants
- Prize pool
- Workshops

### Events Section
Shows event cards for:
- Hackathon
- Webathon
- Coding contest
- Robotics events
- AI workshops

### Event Schedule
Timeline showing the schedule of the festival.

### Speakers Section
Cards displaying:
- Speaker photo
- Name
- Designation
- Organization

### Gallery
Image grid showcasing previous event photos.

### Registration
Form allowing users to register for specific events.

### Contact
Displays venue, contact information, and map location.

## 5. Non-Functional Requirements

### Performance
Pages must load quickly and animations should be smooth.

### Responsiveness
Website must work on:
- Desktop
- Tablet
- Mobile

### Accessibility
Content must be readable with clear typography and color contrast.

## 6. Success Criteria

The project will be considered successful if:

- The website visually resembles the reference site
- All sections are functional
- UI animations work smoothly
- Website is fully responsive
